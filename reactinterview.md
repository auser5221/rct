Reducer function: It is a pure function that takes current state and action as arguments and returns new state. It contains logic for how state should transition in response to different actions

Dispatch function: It is used to send actions to reducer function which then handles state updates based on the actions. Returned by useReducer that you call with an action object. It is only way to trigger a state change.

Passing initial state as an object allows for managing multiple related state variables within a single state structure
